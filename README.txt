# ERP Core - Sistema de Gestão Completo

Sistema ERP desenvolvido em C# com ASP.NET MVC, Web API e WPF.

## Funcionalidades

- ✅ Gestão de Produtos
- ✅ Controle de Estoque
- ✅ Sistema de Autenticação
- ✅ API RESTful
- ✅ Interface Web MVC
- ✅ Aplicação Desktop WPF
- ✅ Sincronização de Dados

## Tecnologias

- .NET 8.0
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- WPF (.NET 8)
- JWT Authentication
